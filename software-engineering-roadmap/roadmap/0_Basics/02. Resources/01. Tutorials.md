# Recommended Tutorials

Here are some recommended tutorials to help you get started with the basics of computer science.

## General Programming
- [Codecademy: Learn Python](https://www.codecademy.com/learn/learn-python-3)
- [FreeCodeCamp: JavaScript Basics](https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures/basic-javascript/)

## Computer Basics
- [Khan Academy: Computer Science](https://www.khanacademy.org/computing/computer-science)
- [Coursera: Introduction to Computer Science by Harvard](https://www.edx.org/course/cs50s-introduction-to-computer-science)

## Operating Systems
- [Linux Journey](https://linuxjourney.com/)
- [Learn Command Line from The Odin Project](https://www.theodinproject.com/paths/foundations/courses/foundations/lessons/command-line-basics)
